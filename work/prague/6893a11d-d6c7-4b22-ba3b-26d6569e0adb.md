## 17시 2분 45초

**User**: 피그마 파일의 annotation도 읽을 수 있어?

**Assistant**: 먼저 이미 가져온 데이터에 annotation이 있는지 확인해볼게요.
현재 Figma MCP 도구로 가져온 데이터에는 **annotation이 포함되어 있지 않습니다.**

Figma의 annotation(Dev Mode에서 추가하는 메모/마크업)은 Figma REST API에서 별도 엔드포인트로 제공되는데, 현재 사용 가능한 `get_figma_data` 도구는 **레이아웃, 컴포넌트, 텍스트, 스타일** 등 시각적 구조 데이터만 반환합니다.

읽을 수 있는 것:
- 노드 계층 구조, 레이아웃
- 텍스트 내용, 텍스트 스타일
- 컴포넌트 정보, 이미지
- fills, strokes, borderRadius 등 스타일 속성

읽을 수 **없는** 것:
- Dev Mode annotations
- Comments (Figma 댓글)
- Version history

Annotation 내용이 필요하시면 Figma에서 직접 확인하시거나, 내용을 텍스트로 공유해주시면 제가 반영할 수 있습니다.
