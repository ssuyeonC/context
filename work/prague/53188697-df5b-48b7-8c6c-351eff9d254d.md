- 좋아요 버튼은 이미 모든 카드에 인라인 노출 + 툴팁까지 있음 → **접근성/인지 문제가 아님**

**전략 변경:**

| 변경 전 | 변경 후 |

| P1: 좋아요 버튼 접근성 강화 | **삭제** (이미 달성된 상태) |
| P1: 좋아요 기반 CRM | **유지** — 기존 1%(~4만명) 대상 시의성 있는 리마인더 |
