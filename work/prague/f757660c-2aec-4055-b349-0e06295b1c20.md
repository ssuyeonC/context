### 적용 로직

| 환불 정책 설정 | 환불 금액 | 취소 가능 여부 |
| --- | --- | --- |
| 100% 환불 가능 구간 | 100% 환불 | 취소 가능 (전액 환불) |
| 부분 환불 구간 | 일부 환불 (예: 50%) | 취소 가능 (부분 환불) |
| 환불 불가 구간 | 0원 | 취소 불가 |
| 환불 정책 미설정 상품 | — | 취소 불가 |

**핵심 원칙**

- 환불 정책상 환불 가능 구간 내 → 취소 가능 (무료 예약이라 실제 환불액이 0원이어도 취소 가능)
- 환불 정책상 환불 불가 구간 → 취소 불가
- 환불 정책 미설정 → 취소 자체가 불가능

즉, "실제 환불 금액"이 아니라 "환불 정책상 어느 구간에 속하는지"가 취소 가능 여부의 기준.

**예시**

Case 1. 무료 예약 상품, "3일 전 100% 환불" 정책 설정 시

- 예약일 4일 전 취소 요청 → 100% 환불 구간 (실제 환불액은 0원이지만 정책상 환불 가능 구간) → 취소 가능
- 예약일 2일 전 취소 요청 → 환불 불가 구간 → 취소 불가

Case 2. "0일 전 100% 환불" 정책 설정 시

- 예약 당일 취소 요청 → 100% 환불 구간 → 취소 가능 (당일 무료 취소 가능)
