**Why it matters**: 어드민 개선 우선순위는 명확히 *예약 처리 효율* + *스팟/블로그 편집기 UX* 두 갈래로 나뉜다. 다른 도메인(`/onlinecoupons`, `/banner`, `/category` 등)은 합쳐도 5% 미만.

## 인사이트 3. "저장" 버튼이 짧은 시간에 중복 클릭되는 비율이 비정상적으로 높다

`label = '저장'` 같은 세션·페이지 내 연속 클릭의 간격 분포 (3,373건):

| 간격 | 건수 | 비중 |
|------|------|------|
| **<2초** | 1,065 | **31.6%** |
| 2~10초 | 611 | 18.1% |
| 10~60초 | 800 | 23.7% |
| 1~5분 | 580 | 17.2% |
| 5분 이상 | 317 | 9.4% |

- 2초 이내 재클릭이 31.6%. 사용자는 저장이 됐는지 즉각적인 피드백을 받지 못해 **반복 클릭**하고 있다고 강하게 의심된다.
- "저장" 클릭이 가장 많은 페이지 5개 중 **4개가 블로그 에디터**(`/blog/14605` 627, `/blog/13720` 477, `/blog/3001` 419, `/blog/2487` 335). 블로그 편집기의 저장 피드백을 점검 필요.
- 동일 패턴이 `예약 확정하기 (1,382 클릭, 280세션 = 1인당 약 5회)`, `특이사항 저장 (1,315 클릭, 106세션 = 1인당 약 12회)`에서도 보임 — 처리 후 응답 지연 또는 페이지 리로드 부재.

**Why it matters**: "저장이 됐는지 불안한 UX"는 즉시 개선 가능. 저장 시 토스트/디스에이블/스피너 보강만으로도 클릭 노이즈 30% 절감 + 사용자 불안 감소.

## 인사이트 4. 동일 예약 행(`tr`)을 한 세션에서 100회 이상 클릭하는 패턴이 존재

`/reservation` 페이지에서 `element_type='tr'` 클릭 상위 행들이 **세션당 144 클릭** 같은 일관된 숫자를 보임:

```
260503yfv42e LANE4 공항픽업 ... — 147 클릭, 145 세션 (≒1세션당 1번씩)
2601128io85k LANE4 합승형 ... — 146 클릭, 146 세션
... (Lane4 LANE4 행 다수)
```

- 모두 동일 파트너(Lane4)의 예약 행. 한 행이 매 세션마다 한 번씩 열림 → 외부 파트너가 **자기 예약 목록을 매 세션 처음부터 다시 검토**하고 있을 가능성.
- 또는 외부 파트너 어드민이 **목록을 닫고 다시 여는 동작 자체가 발생**하고 있음을 시사. (`Close 4,473`도 동일 맥락)

**Why it matters**: 파트너가 같은 예약을 반복 조회하는 이유(미처리 표시 없음? 상태 변경 후 다시 확인? 모달이 list 위 덮여 매번 다시 클릭?)를 정성 조사할 가치가 크다. "이미 처리한 예약 숨기기", "스티키 필터", "디테일 인라인 표시" 등이 후보.

## 인사이트 5. 어드민 페이지 곳곳에 `label_source='fallback'`이 광범위하게 발생 → 트래킹·접근성 동시 결함

| label_source | 비중 |
|--------------|------|
| text-content | 78% |
| aria-label | 6.7% |
| icon | 5.4% |
| label-element | 3.7% |
| **fallback** | **3.5%** (4,951건) |
| placeholder | 1.5% |

- fallback의 88%는 `<input>` (4,303건). 라벨이 없는 입력 필드가 광범위.
- 가장 많은 페이지가 모두 스팟 신규 상품 페이지(`/spot/15499` 158, `/spot/15507` 118 등). **신규 상품 등록 폼에 라벨 없는 input 다수 존재**.
- `track_breadcrumb`/`aria_label` 컬럼은 어드민 전 구간에서 0건 — 어드민에서는 클라이언트 트래커가 이 두 필드를 채우지 않는 것으로 보임. **트래커 설정 자체에 격차가 있다**.

**Why it matters**: (a) 트래킹 품질이 낮아 향후 더 정교한 분석이 불가, (b) `<label>`/`aria-label` 누락은 접근성 문제이기도 함. 신규 상품 등록 폼 우선으로 input 라벨 일제 점검 권장.

## 인사이트 6. `icon:clock-circle` 등 의미 불명 아이콘이 단일 어드민·단일 페이지에서 폭발적으로 클릭됨

**Why it matters**: 시간/옵션 편집기에 단축키 또는 일괄 처리(예: 요일·시간대 일괄 적용)가 있으면 반복 클릭을 크게 줄일 수 있다. 어떤 옵션 편집 워크플로가 가장 비효율적인지 영상 관찰로 검증할 가치 있음.
