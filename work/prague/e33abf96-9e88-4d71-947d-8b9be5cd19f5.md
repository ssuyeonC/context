## 안 되면 수동 정리

캐시된 토큰을 직접 제거하고 다시 인증하는 방법입니다.

```bash
# 1) 현재 등록된 MCP 서버 확인
claude mcp list

# 2) creatrip 서버 제거
claude mcp remove creatrip

# 3) 다시 추가 (URL은 기존 등록 명령 그대로)
claude mcp add creatrip <원래-URL> --transport http
```
