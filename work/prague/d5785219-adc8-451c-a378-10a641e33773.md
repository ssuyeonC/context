- 타입은 카테고리(`CATEGORY`) / URL(`URL`) 중 택일
  - `CATEGORY` 타입: 대분류 카테고리(MAIN_RESERVATION)와 연결, 이름과 아이콘은 카테고리의 `translation.name` 및 `mapIconUrl`을 참조
  - `URL` 타입: 이름 직접 입력, 아이콘(SVG) 직접 업로드, URL 직접 입력
- `URL` 타입의 아이콘은 복수 SVG를 지원 (2개 이상 시 유저 페이지에서 캐러셀 렌더링)
- 각 항목은 노출 순서값을 가짐

**home_navigation_badge (신설)**
- 내비게이션 뱃지 엔티티 신설
- 뱃지 타입은 `HOT`, `NEW`, `BEST` 3종으로 사전 정의 (CRUD 불가)
- 각 뱃지는 언어별로 최대 1개의 내비게이션 항목과 연결 가능
- 언어별 독립적 연결 (예: 한국어 BEST → 항목A, 일본어 BEST → 항목B)
