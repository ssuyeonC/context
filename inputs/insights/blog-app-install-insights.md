# 블로그 도메인 인사이트 — 앱 다운로드 전환 관점

## 1. 배경

앱 다운로드 유저 수를 늘리기 위한 액션을 검토하는 과정에서, 페이지별 활성 사용자 규모를 비교한 결과 **블로그 상세 페이지(/blog/{id})가 538K 활성 사용자로 압도적 1위**임을 확인. 블로그를 앱 설치 유도의 주요 채널로 활용하는 방향을 탐색.

### 검토했으나 기각한 안: 메인홈 → 지도(/map) 유도
- 메인홈 모바일웹(한국) 활성 사용자: 1,253명/월
- /map 활성 사용자: 4,916명/월 (GA 기준)
- 퍼널 단계가 많아(메인홈 → 피드 클릭 → /map → POI 클릭 → 앱 설치) 월 2~6명 수준의 전환 예상 → 임팩트 부족으로 기각

## 4. 블로그의 두 가지 성격 (정성 인사이트)

### 여행 외 정보 블로그 (한국 이름, 연예 정보 등)
- **조회수가 압도적으로 많음**
- 스팟이 연결되어있는 경우도 있으나, **스팟 전환율이 매우 낮음**
- 유입 목적 자체가 여행이 아니라 정보 소비

### 여행 상품 관련 블로그
- 조회수는 상대적으로 높지 않음
- **스팟 전환율(예약 연결률)이 높음** — MIDDLE_RESERVATION의 경우 99.7%
- 이미 여행 의도가 있는 유저가 유입

- [ ] **블로그 페이지의 모바일 웹 비율**: GA4 접근 필요 (현재 토큰 만료)
