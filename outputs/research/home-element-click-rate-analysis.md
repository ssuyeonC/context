# 메인 홈 요소별 클릭율 분석

> **출처**: 2026-04-27 세션 (`.context/work/prague/e33abf96-9e88-4d71-947d-8b9be5cd19f5.md`)에서 정리
> **분석 기간**: 2026-04-20 ~ 2026-04-27 (최근 7일)
> **데이터 소스**: 어플리케이션 DB `general_click_event` (DOM 자동 클릭 트래킹) + `blog_translation`
> **도구**: dba-mcp 직접 쿼리 (BigQuery `dashboard_ga4`/`processed`/GA4 raw export 모두 권한·인증 이슈로 불가했던 차선책)

---

## 1. 분석 목적

메인 홈(`/`)에 노출되는 각 요소(카테고리 내비, 진입점 카드, 아코디언 서브메뉴, 블로그 카드 등)가 실제 사용자 클릭을 얼마나 받는지 정량 파악.

특히 **상품형 컨텐츠 vs 정보형 컨텐츠**의 클릭 분포를 비교하여 "여행 정보·가이드 중심 컨텐츠에 클릭이 더 몰리는가"라는 가설을 데이터로 점검.

---

## 2. 산출 방식과 한계 (먼저 읽을 것)

### 데이터 한계

| 항목 | 상태 |
|---|---|
| 임프레션 (노출) 데이터 | **DB에 없음** — `BlogCardImpression`/`BlogCardClick` 등 GA 이벤트는 `dashboard_ga4`에 집계 테이블 없음. `processed.*` / `analytics_*.events_*` 권한 없음. GA4 Data API는 `invalid_grant`로 인증 만료 |
| 진짜 CTR (클릭 ÷ 임프레션) | **산출 불가** |
| 차선책 | **"홈에서 클릭이 1회 이상 발생한 세션 중 해당 요소를 누른 세션 비율"** 사용 |

### 디바이스 구분 한계

- `general_click_event.user_agent`는 모든 행이 `'Amazon CloudFront'` (CloudFront가 백엔드로 요청 전달 시 UA를 자기 값으로 덮어씀)
- BigQuery `dashboard_ga4.HomeBanner`의 `device_category`는 GA4 `ga_session_id`(INT64)와 우리 `general_click_event.session_id`(UUID)가 다른 식별자라 조인 불가
- **휴리스틱 분류**: 모바일/태블릿 전용 DOM(`<details>/<summary>` 아코디언) 클릭 유무로 구분
  - **확정 모바일**: 홈에서 `element_type='summary'` 클릭이 있는 세션
  - **확정 데스크톱**: Home-SubMenu 안쪽 `<a>`/`<button>` 클릭 + `<summary>` 없음
  - 데스크톱 탭 버튼은 `<div>` + onClick 구조라 `general_click_event`에 잡히지 않음 → **데스크톱 표본 작음**
- 결과: 확정 모바일 2,623 / 확정 데스크톱 339 / Ambiguous 13,512

### 해석상 caveat

- 임프레션이 없어 "홈에 들어왔지만 아무 클릭도 안 한 세션"은 분모에서 누락 → **이미 인게이지된 사용자의 행동만 본 것**
- 같은 세션이 여러 요소를 클릭할 수 있음 → 세그먼트(상품 관심자 vs 정보 관심자) 분리 검증은 별도 코호트 분석 필요

## 4. 시사점 (가설 단계)

> 다음은 데이터로 **시사**되는 패턴이지, 인과나 사용자 세그먼트 분리를 증명한 것은 아님.

1. **트렌드/큐레이션 컨텐츠가 압도적 우위**
   - HOT 트렌드 24.64%, 다른 모든 진입점·아코디언을 합쳐도 견주기 어려움
   - 메인 홈에서 "지금 인기 있는 것"을 빠르게 보여주는 진입점이 가장 유효한 핵심 동선

2. **정보형 컨텐츠가 카테고리 진입점보다 클릭이 많음**
   - 환율 아코디언(11.01%)이 모든 카테고리 진입점·진입 카드를 합한 수치보다 큼
   - 블로그 카드(3.51%)도 KBeauty/Travel/Stay 진입점 단독 수치보다 높거나 비슷
   - **단, 이는 "정보 관심 사용자가 더 많다"가 아니라 "정보 컨텐츠의 진입 장벽이 더 낮다(=같은 화면 내 펼침)"가 원인일 수도 있음**

3. **상품 카테고리 진입점은 분산되어 있어 단독 수치가 작게 보임**
   - Travel(헤더 2.50%) + KBeauty(3.03%) + LanguageCourse(1.63%) + Stay(1.61%) + 카테고리 아이콘(3.04%) + ALL버튼(7.71%) = 약 20%
   - 합산 시 HOT에 근접 → **"상품 관심이 적다"가 아니라 분산되어 시인성이 떨어진다**고 해석 가능

4. **블로그 카드: 핀 ≠ 클릭 1위**
   - 핀 처리된 블로그(호수도산, 옵틱라이프)는 동률 3위
   - 1·2위는 여행 일반 정보 가이드 → **사용자의 실제 의도는 광고성 블로그보다 본질적 여행 정보 쪽**

## 6. 데이터 caveat 다시 한번

- **모든 수치는 "이미 클릭한 세션" 기준 비율** — 페이지 노출 대비 클릭율(진짜 CTR)이 아님
- **모바일/데스크톱 분리는 휴리스틱** — 데스크톱 확정 표본은 339로 작음, Ambiguous에 모바일·데스크톱 혼재
- **세션 모집단(분모)에 따라 수치가 크게 달라짐** — 분모 변경 시 항상 명시할 것
