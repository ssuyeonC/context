## 인터뷰 후 정리 템플릿

| 차원 | 항목 | 메모 |
| --- | --- | --- |
| 행동 | 누가 / 언제 / 어디서 / 어떻게 (구체 동작) | |
| 워크어라운드 | 어드민에 없어서 카톡·전화로 처리하는 것 | |
| 시간/사람 | 작업당 분, 사람 수, 처리 주기 | |

특별히 추출해야 할 것:
- **카톡→어드민 전환 트리거** (다른 파트너 온보딩 가설 인풋)
- **시간슬롯 vs 콘텐츠 수정의 채널 선택 메커니즘** (영역별 양극단 활용 사유)
- **메디컬 카테고리 특수 마찰** (시술명·가격·의료광고 규정)
- **본사(에이피알) ↔ 지점 권한 분담 모델** (체인 의료기관 일반화)
- **어드민에 없지만 필요한 기능 후보** (우회 경로에서 발견)
