# 앱 전용 쿠폰 혜택 설계 보고서

## 목적

웹 유저를 앱 유저로 전환하기 위한 앱 전용 쿠폰의 할인 금액·최소 주문·유효기간 스펙 결정.

---

## 배경: 왜 앱 유저를 늘려야 하는가

DAU 모델 분석 결과, 앱 유저는 **모든 버킷에서** 웹 유저보다 구매 전환율이 높다.

| 버킷 | 웹 CVR | 앱 CVR | 배수 |
|---|---|---|---|
| NEW USER | 1.14% | 4.94% | 4.3배 |
| RETURN_D1 (합산) | 2.77% | 6.36% | 2.3배 |
| RETURN_D7 (합산) | 2.18% | 4.61% | 2.1배 |
| RETURN_D30 | 1.88% | 3.50% | 1.9배 |
| RESURRECTED | 1.52% | 3.11% | 2.0배 |

또한 앱은 리텐션 버킷(RETURN_D1) 비중이 웹의 2배 수준으로, 고전환 버킷에 유저가 더 오래 머무르는 구조적 우위가 있다.

---

## 분석: 웹→앱 전환의 증분 가치

### 유저 1명의 1회 방문당 기대매출 (CVR × 인당 거래액)

| 버킷 | 웹 기대매출 | 앱 기대매출 | 증분 가치 |
|---|---|---|---|
| NEW USER | 2,135원 | 10,810원 | +8,675원 |
| D1&NEW | 6,658원 | 18,523원 | +11,865원 |
| D1 | 5,340원 | 12,386원 | +7,046원 |
| D7&NEW | 4,949원 | 9,420원 | +4,471원 |
| D7 | 3,929원 | 8,494원 | +4,565원 |
| D30 | 3,531원 | 6,758원 | +3,227원 |
| RESURRECTED | 2,777원 | 4,179원 | +1,402원 |

웹 유저 버킷 분포로 가중 평균한 **1회 방문당 증분 가치: ~6,679원**.

### 연간 증분 가치 추정

전환된 웹 유저가 기존 앱 유저의 50% 수준으로만 행동한다고 보수적으로 가정:

| | 웹 유저 | 앱 전환 유저 (50% 보정) |
|---|---|---|
| 월 방문 | 1.5회 | 2회 |
| 방문당 기대매출 | 3,228원 | 4,577원 (9,153원 × 50%) |
| **연간 기대매출** | **~58,104원** | **~109,836원** |

**연간 증분 가치: ~51,732원**

## 쿠폰 금액 결정

### 프레이밍

이 쿠폰의 비용은 "1회 거래 할인"이 아니라 **"앱 유저 획득 비용(CAC)"**이다.

- 발급만으로는 비용이 발생하지 않음 (사용 시에만 비용)
- 원래 구매 의도가 있던 웹 유저도 쿠폰 없이는 앱을 설치하지 않았을 것
- 이런 유저가 앱으로 전환되면 이후 방문에서도 앱 유저로 행동 → 장기 가치 발생

### 금액별 비교

| 쿠폰 금액 | 연간 증분 가치 대비 | 앱 설치 동기 | 운영 여지 |
|---|---|---|---|
| 3,000원 | 5.8% | 약함 | 올리기 쉬움 |
| **5,000원** | **9.7%** | **충분** | **균형점** |
| 6,000원 | 11.6% | 강함 | 내리기 어려움 |

- **3,000원**: 앱 설치라는 허들을 넘기기에 동기 부족 우려
- **5,000원**: 심리적으로 명확한 숫자, 증분 가치 대비 충분한 마진, 추후 최적화 여지 확보
- **6,000원**: 정당화 가능하나, 금액을 낮추는 방향의 조정이 어려움

### 유효기간: 없음

- 여행 구매는 타이밍이 유저마다 다름 → 유효기간이 전환 기회를 차단할 수 있음
- 쿠폰이 살아있어야 결제 페이지에서 "앱에서 5,000원 할인" 안내가 지속적으로 유효
- 미사용 쿠폰은 비용이 아님

## 결론: 스펙

| 항목 | 스펙 |
|---|---|
| 할인 금액 | **5,000원** |
| 최소 주문 금액 | **30,000원** |
| 유효기간 | **없음** |
| 발급 대상 | 회원가입 시 자동 발급 (기존 회원 일괄 발급) |
| 발급 횟수 | 1인 1회 |
| 사용 조건 | 앱 환경에서만 적용 가능 |

---

## 추후 검토 사항

- 앱 설치 전환율에 대한 금액별 A/B 테스트 (3,000원 vs 5,000원)
- 전환된 유저의 실제 행동 데이터 추적 (기존 앱 유저 대비 CVR, 리텐션)
- 기존 쿠폰과의 중복 사용 정책
