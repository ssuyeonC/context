# 세부 기획안 요구사항 문서 양식

## 배경
### 문제 정의
파트너 어드민의 '파트너쉽 정산' 페이지(admin.creatrip.com/travel-partner/settlements)의 현재 UI가 정산 현황을 한눈에 파악하기 어려운 구조로 되어 있음.

- 누적 정산액이 단순 텍스트로 표시되어 시각적 강조가 부족함
- 현재 월/다음 월 미정산액을 별도로 확인할 수 없어 파트너가 정산 일정을 예측하기 어려움
- 기간별 필터링 및 정산 상태(예정/완료) 필터가 없어 원하는 데이터를 탐색하는 데 불편함
- 테이블에서 정산 여부를 직접 확인할 수 없어 별도 확인 과정이 필요함

### 가설적 임팩트
- 파트너가 정산 현황을 직관적으로 파악할 수 있어 정산 관련 문의 감소 예상
- 필터 기능 추가로 데이터 탐색 효율성 향상, 파트너 어드민 사용 만족도 개선
- 정산 여부 열 추가로 파트너의 정산 상태 확인 프로세스 단축

## 구현
### 유저 플로우
1. 파트너는 정산 페이지에 접근하여 카드 형태의 누적 정산액 영역에서 총 누적 정산액, 현재 월 미정산액, 다음 월 미정산액을 한눈에 확인할 수 있다.
2. 파트너는 기간 필터를 사용하여 원하는 기간의 정산 내역만 조회할 수 있다.
3. 파트너는 정산 상태 필터(정산 예정 / 정산 완료)를 사용하여 원하는 상태의 내역만 조회할 수 있다.
4. 파트너는 테이블의 '정산 여부' 열을 통해 각 내역의 정산 상태(정산 예정 / 정산 완료)를 바로 확인할 수 있다.

### 프론트엔드 요구사항
- **누적 정산액 영역 UI 변경**: 기존 텍스트 표시를 카드 형태의 UI로 변경
  - 누적 정산액 카드 (메인)
  - 현재 월(N월) 미정산액 카드: `N월 정산 예정액` 레이블과 금액 표시
  - 다음 월(N+1월) 미정산액 카드: `N+1월 정산 예정액` 레이블과 금액 표시
- **기간 필터 추가**: 날짜 범위 선택(YYYY/MM/DD ~ YYYY/MM/DD) 필터 컴포넌트 배치
- **정산 상태 필터 추가**: `정산 예정` / `정산 완료` 탭 또는 필터 버튼 추가
- **필터 및 정렬 컴포넌트 위치 조정**: 기존 필터/정렬 UI의 레이아웃 재배치 (디자인 시안 참고)
- **테이블 '정산 여부' 열 추가**: 각 행에 `정산 예정` 또는 `정산 완료` 상태 표시
- 대상 페이지: `admin.creatrip.com/travel-partner/settlements`

### 백엔드 요구사항
- **미정산액 조회 API**: 현재 월 미정산액, 다음 월 미정산액을 반환하는 API 제공 (또는 기존 API 응답에 필드 추가)
- **기간 필터 대응**: 정산 내역 조회 API에 기간(startDate, endDate) 파라미터 지원
- **정산 상태 필터 대응**: 정산 내역 조회 API에 정산 상태(settlementStatus: SCHEDULED | COMPLETED) 파라미터 지원
- **정산 여부 필드**: 정산 내역 응답에 정산 여부(settlementStatus) 필드 포함
