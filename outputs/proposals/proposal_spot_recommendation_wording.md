# 스팟 상세페이지 추천 영역 개선 제안

### 왜 이게 문제라고 생각했는가?

- 스팟 상세페이지 내 섹션 이동 탭에서 **"Recommendations" 버튼의 클릭률이 모든 세그먼트에서 현저히 낮음**
    - 섹션 이동 탭 전체 클릭률: 약 **3.25%** (DetailPageSectionClick / DetailPageSectionImpression)
    - 그 중 Recommendations 탭 클릭률: **3.9%** 수준으로, Options·Reviews·StoreInfo 등 다른 탭 대비 가장 낮음
    - 데스크탑에서 기본 노출 상태임에도 불구하고 클릭률이 낮음
- 그러나 **추천 섹션 내 스팟 카드 자체의 클릭률은 약 13.8%** (SpotProductCardClick / 전체 스팟 PV)로, 낮은 수치가 아님
    - 즉, 추천 카드의 품질이 아니라 **"Recommendations"라는 워딩의 매력도**가 문제
- 기존 가설이었던 "추천 영역이 너무 아래에 있어서 문제"는 재검토가 필요
    - 유저의 탭 클릭 패턴을 보면 **구매 의도(Options)와 학습(Reviews & StoreInfo)** 관련 탭에 의도가 집중됨
    - 상품 탐색 흐름 중간에 다른 상품 추천이 노출되면 오히려 **노이즈로 인식될 가능성**이 있음
- 추가로, 추천 로직 자체에도 개선 여지가 있음
    - "Frequently Booked Together": 시간 제한 없이 전체 구매 이력을 조회 → 다른 여행의 상품이 섞여 맥락이 희석됨
    - "Products Viewed by Other Customers": 카테고리·지역 구분 없이 조회 빈도순 → 대체재로서의 역할이 명확하지 않음

### 어떤 사용자에게 영향이 있을까?

- **스팟 상세페이지 방문자 (New User 비중 높음)**
    - 스팟 상세페이지 트래픽의 대부분을 차지하는 세그먼트
    - 개인화 데이터가 없는 상태에서 추천 품질과 접근성이 탐색 경험의 핵심
    - DAU 모델 분석 기준, New User → D1 Return User 전환이 핵심 지표이며, "여행 상세페이지 하단 추천 상품 클릭"은 **리텐션과 구매 전환율을 모두 높이는 행동**으로 확인됨
- **여행 준비 중인 유저 (세그먼트 1: 여행 전 × 스팟/숙소 상세)**
    - "이 상품 외에 뭘 더 준비해야 하지?"라는 질문에 답을 주는 영역
    - 보완재 추천이 잘 작동하면 추가 탐색 → 재방문 동기 부여로 이어짐
- **비교 탐색 중인 유저**
    - 같은 카테고리 내 대안을 찾는 유저에게 "Compare Alternatives" 영역이 의사결정을 도움
    - 현재는 카테고리·지역 구분 없이 노출되어 비교 가치가 떨어지는 상태

### 가설적 임팩트?

- **왜 교차 탐색을 개선해야 하는가?**
    - 내부 데이터 분석 결과, **"여행 상품을 2개 이상 탐색"한 유저**는 D1 리텐션 확률과 구매 전환율이 모두 높아지는 것으로 확인됨
    - 즉, 추천 영역을 통해 유저가 자연스럽게 다른 스팟을 탐색하도록 유도하는 것 자체가 리텐션·구매 전환 모두에 기여하는 레버
    - 현재 추천 탭의 낮은 진입률(3.9%)은 이 교차 탐색 기회를 놓치고 있다는 의미이며, 이를 개선하는 것이 이번 제안의 핵심 목표
- **액션 1 — 워딩 변경 (탭 진입량 확보)**
    - Recommendations 탭 클릭률 **3.9% → 개선** 목표
    - 워딩만으로도 유저의 탭 진입을 유도할 수 있다면, 추천 카드 클릭(13.8%)과 결합하여 교차 탐색 증가 기대
    - "여행 상세페이지 하단 추천 상품 클릭"이 D1 리텐션 + 구매 전환 모두에 긍정적 영향을 주는 행동으로 확인된 만큼, **탭 진입 자체를 늘리는 것**이 이 단계의 핵심
    - 주요 지표: Recommend 탭 클릭률
    - 가드레일: 다른 탭 클릭률 유지, 카드 클릭 품질(13.8%) 유지, 구매 전환율
- **액션 2 — 로직 데이터 정제 (추천 품질 기반 마련)**
    - 액션 1에서 탭 유입이 늘어난 상태에서, 유입된 유저에게 **최근 여행 맥락에 맞는 추천**을 보여주기 위한 준비 단계
    - time_decay·계절성 반영 + 30일 이내 데이터만 반영하여 "다른 여행에서 산 상품"이 섞이는 문제를 제거
    - 이 단계 자체는 A/B 테스트 없이 로직 개선으로 진행하되, 액션 N에서의 효과 측정 기반이 됨
- **액션 N — 타이틀 + 로직 변경 A/B 테스트 (액션 1·2 이후, 카드 클릭 품질 개선)**
    - 액션 1로 탭 유입이 확보되고, 액션 2로 데이터가 정제된 상태에서 진행
    - 보완재(Complete Your Trip): 카테고리 간 교차구매 기반으로 "여행 준비가 더 필요하다"는 인식 유도
    - 대체재(Compare Alternatives): 같은 카테고리 + 지역 기반으로 비교 가치 향상
    - 테스트 결과, 보완재 로직에서 카테고리 간 겹침 없이 자연스러운 구성이 확인됨
        - 순시키헤어 → 피부시술/포토/증명사진
        - 롯데월드 → 교복대여/헤어/증명사진
        - 진미식당 → 한복대여/헤어/증명사진
    - 주요 지표: 추천 섹션 내 스팟 카드 클릭률
    - 가드레일: 구매 전환율, 세션당 스팟 상세 조회 수, 액션 1에서 확보한 탭 클릭률 유지
- **종합 기대 효과**
    - 액션 1(유입) → 액션 2(데이터 정제) → 액션 N(추천 품질)으로 이어지는 순차 구조
    - 각 단계가 다음 단계의 전제 조건이 되므로, 단계별로 효과를 측정하고 다음 액션 진행 여부를 판단할 수 있음
    - 장기적으로 D1 Retention 및 교차 탐색 → 구매 전환율 개선에 기여할 것으로 기대

### 해결 방법 논의 및 실행

1. **[액션 1] 섹션 탭 "Recommend" 워딩 변경 A/B 테스트**
    - 일정: 02/20 코어 유닛 내 공유 후 바로 진행
    - 변경: "Recommend" → **"Booked Together"** (+ 기타 수정안)
    - 주요 지표: Recommend 탭 클릭률 (현재 3.9%)
    - 가드레일 지표:
        - 다른 탭(Options, Reviews 등) 클릭률 감소 여부
        - 추천 섹션 내 스팟 카드 클릭률(13.8%) 유지 여부
        - 구매 전환율 영향 여부

2. **[액션 2] Frequently Booked Together 로직 개선**
    - time_decay 및 계절성 반영, **30일 이내** 구매 데이터만 추천에 반영
    - 기존: 시간 제한 없이 전체 구매 이력 조회
    - 개선: 최근 여행 맥락에 맞는 추천으로 품질 향상

3. **[액션 N] 두 추천 영역 로직 & 타이틀 변경 A/B 테스트 (액션 1 이후)**
    - **Frequently Booked Together → "Complete Your Trip" / "Pair Well With This" (보완재)**
        - 로직 개선:
            1. 대상 스팟의 MIDDLE_RESERVATION / MAIN_RESERVATION 카테고리 파악
            2. 해당 스팟을 완료/사용 상태로 구매한 유저의 교차 구매 데이터 조회
            3. 필터링: 요청 카테고리 포함, 대상 스팟의 MIDDLE 카테고리 제외, 활성 상품만
            4. 정렬: 교차 구매 빈도 DESC (특수: 헤어 → 피부과 우선 정렬)
            5. Fallback: 교차 구매 부족 시 → 함께 본 스팟(Watch-Together) → 기존 로직
    - **Products Viewed by Other Customers → "Compare Alternatives" (대체재)**
        - 로직 개선:
            1. 대상 스팟의 MIDDLE_RESERVATION 카테고리 및 지역 카테고리 파악
            2. 같은 MIDDLE 카테고리 내 교차 구매 조회
            3. 지역 Fallback: DETAIL_LOCATION(같은 동네) → CITY(같은 도시) → 지역 무관, 15개 채우면 중단
            4. 정렬: 교차 구매 빈도 DESC
    - 주요 지표: 추천 섹션 내 스팟 카드 클릭률
    - 가드레일 지표: 구매 전환율 / 세션당 스팟 상세 조회 수 / 1차에서 확보한 탭 클릭률 유지
