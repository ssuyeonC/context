# K-Beauty 페이지 리디자인 제안서

> **작성일**: 2026-03-26
> **작성**: Product팀
> **상태**: 내부 리뷰 대기
> **관련 문서**: Notion 원문 · 3/22 개선 방향 회의록 · 3/24 데이터 리뷰 · 3/25 아이데이션 회의록 · 외국인 피부과 인지도 리서치

---

### 왜 이게 문제라고 생각했는가?

K-Beauty 페이지는 뷰티 전략 카테고리의 **유일한 허브 페이지**이지만, 현재 구조가 타겟 유저의 의사결정 흐름과 맞지 않습니다.

**현재 퍼널 (3/24 데이터 기준)**

```
K뷰티 페이지 조회      873 세션/월 (일 약 30)
       │
       │  18.9% 스팟 클릭
       ▼
스팟 카드 클릭          165 세션
       │
       │  7.9% 구매
       ▼
스팟 구매 완료           13 세션 (CVR 1.49%)
```

- **81%의 유저(708명)가 스팟을 클릭하지 않고 이탈** — 가장 큰 손실 구간
- 시술 추천(8단계 모달) CTR은 **1.4%** — 사실상 사용되지 않는 기능
- 같은 세션 CVR 1.49%는 고관여 상품 특성상 과소 측정 가능성 있음 (크로스 세션 시 32.9%가 스팟 상세 방문)

**구조적 문제**

| 문제 | 현재 상태 | 영향 |
|------|---------|------|
| 정보 과부하 | **18개 섹션**이 타입별로 산발 배치 | 유저가 "다음에 뭐가 나올지" 예측 불가, 이탈 유발 |
| 스토리라인 부재 | 블로그 3곳, 리뷰 3곳 분산 | 같은 유형 콘텐츠가 반복 — 유저가 차이를 인지 못함 |
| 추천 진입 장벽 | 8단계 모달, All 탭에서만 노출 | CTR 1.4%, 개인화 경험 사실상 미제공 |
| 기술 부채 | 블로그 배너 14개 언어 하드코딩, Trust 숫자 하드코딩 | 콘텐츠 교체가 개발 배포에 의존 — 마케팅팀 자체 운영 불가 |

페이지 자체의 전환 구조에는 개선할 여지가 있지만, **월 873세션(일 30)이라는 절대 트래픽이 이미 매우 작다**는 점이 핵심입니다. 페이지 내 스팟 클릭율을 18.9%에서 30%로 끌어올린다 해도, 월 기준으로 **추가 유입되는 스팟 클릭은 약 97건** 수준입니다. 이 중 실제 구매로 이어지는 건수는 한 자릿수에 머물 가능성이 높습니다.

---

### 어떤 사용자에게 영향이 있을까?

**"시술을 받기 위해 한국에 오는" 고관여 목적형 방문자**

리서치 결과(Gemini 딥 리서치, 레딧 포스트·야놀자 리서치 등 50+개 소스), 이 페이지를 찾는 유저는 초보자가 아니라 고관여 소비자입니다.

- **전제 1 — 피부 미용이 여행의 '목적'이다**: 피부과 진료가 외국인 환자 방문의 56.6%로 압도적 1위(2024년 117만 명). "한국 여행 온 김에"가 아니라 "시술을 받기 위해 방문"하는 사람들
- **전제 2 — 고객은 이미 '무엇을 받아야 하는지' 안다**: 리쥬란 종류별 차이를 구분하고, 콤비네이션 테라피를 고객이 먼저 제안하는 수준
- **전제 3 — 국가별로 고객 여정이 완전히 다르다**: 일본(가성비), 중국(정품 신뢰), 미국/유럽(과학적 근거), 동남아(K-뷰티 동경) 등 동기와 의사결정 기준이 상이

| 국적 | 핵심 동기 | 의사결정 기준 | 선호 시술 |
|------|----------|-------------|----------|
| 일본 | 가성비 + 빠른 시술 | 가격, 소요 시간 | 보톡스, 레이저, 점 빼기 |
| 중국 | 정품 신뢰 + 프리미엄 | 정품 인증, 의료진 경력 | 울쎄라, 써마지 |
| 미국/유럽 | 과학적 근거 | 성분 원리, 임상 결과 | 리쥬란, 쥬베룩 |
| 동남아 | K-뷰티 동경 | 비주얼 결과, 트렌드 | 리쥬란, 엑소좀 |

다만, **월 873세션이라는 모수를 감안하면 실제 영향을 받는 유저 규모 자체가 제한적**입니다. 페이지 개선이 아무리 효과적이더라도, 이 트래픽 위에서의 전환 개선분은 절대 수치로 크지 않습니다.

---

### 가설적 임팩트?

**페이지 내 전환 개선 — "할 수는 있지만, 모수가 작다"**

| 지표 | 현재값 | 목표값 | 절대 수치 변화 |
|------|-------|-------|-------------|
| K-beauty 내 스팟 클릭율 | 18.9% (165건/월) | 25~30% | +53~97건/월 |
| 시술 추천 참여율 | 1.4% | 5~10% | +31~75건/월 |
| 페이지 이탈율 | 미측정 | 베이스라인 대비 -15% | — |
| 스크롤 깊이 | 미측정 | 50% 이상 유저가 Phase 3 도달 | — |

**솔직한 평가:**

- 스팟 클릭율을 10%p 올려도 **월 87건의 추가 클릭**. 이 중 구매 전환율(현재 7.9%)을 적용하면 **월 6~7건의 추가 구매**로, GMV 기여분은 제한적
- 크로스 세션 전환까지 포함하면 기여가 더 클 수 있으나, 현재 어트리뷰션 측정 체계가 없어 정확한 추정이 어려움
- **진입 트래픽(873세션)이 먼저 올라가야 페이지 내 개선의 효과가 의미 있는 규모로 나타남** — 진입 확대(GNB 승격, 메인 카드 리디자인 등)가 선행되지 않으면 페이지 내 개선만으로는 임팩트가 제한적

**정성적 기대 (장기):**

- 콤비네이션 패키지 발견 경로 확보로 객단가 상승 가능성
- 시술 비교 가이드가 레딧/SNS 공유 포맷과 일치 시 오가닉 유입 경로 확보
- 어드민화로 마케팅팀 자체 운영 가능 — 개발 의존 탈피

**결론적으로, 페이지 구조 개선을 통해 전환을 높일 여지는 분명히 존재하나, 현재 월 873세션이라는 모수 위에서는 절대적인 임팩트가 크다고 보기 어렵습니다. 따라서 이 작업의 우선순위는 높지 않으며, 진입 트래픽 확대 시책이 선행되거나 병행될 때 비로소 의미 있는 효과를 기대할 수 있습니다.**

### 해결 방법 논의 및 실행

만약 진행한다면, 의사결정 흐름 기반의 **4-Phase 10개 섹션** 구조로 재설계하는 방향입니다. 기존 18개 섹션을 축소하고 스토리라인을 부여합니다.

**전체 페이지 구조**

```
Phase 1: 인지 — "여기서 예약하면 이득이다"
├─ [1-A] Hero (비주얼 + 카피 + Social Proof 인라인)
├─ [1-B] Beauty Cashback (독립 섹션)
└─ [1-C] Trust — Why Creatrip (3카드)

Phase 2: 탐색 — "나에게 맞는 시술과 클리닉 찾기"
├─ [2-A] 실시간 인기 시술
├─ [2-B] Embedded Step Wizard (시술→예산→지역)
├─ [2-C] Category Tab + Spots
└─ [2-D] 가격 비교 가이드

Phase 3: 검증 — "실제 결과와 전문성으로 신뢰 확인"
├─ [3-A] 통합 리뷰 (필터 + 국가 플래그 + 시술명 태그)
└─ [3-B] 시술 가이드 (블로그 리프레이밍)

Phase 4: 안내 — "클리닉 찾기와 상담"
├─ [4-A] 지도
└─ [4-B] Inquiry Form

[Floating] 챗봇
```

**Phase별 핵심 설계**

**Phase 1 — 인지**: 목적형 방문자에게 "왜 크리에이트립인가"만 빠르게 전달. Hero에 Social Proof 인라인 통합, 캐시백을 독립 섹션으로 승격, Trust 카드에 정품 인증·투명한 가격 리프레이밍 적용.

**Phase 2 — 탐색 (핵심 구간)**: 스팟 클릭율 18.9% → 25~30% 달성이 목표인 구간.
- 실시간 인기 시술을 최상단 배치 (순위 기반, 콤비네이션 태그 지원)
- 기존 8단계 모달을 3단계 인터랙티브 탐색(시술→예산→지역)으로 간소화
- Category Tab + Spots는 기존 유지 (직접 탐색 경로)

**Phase 3 — 검증**: 리뷰 3개 섹션을 하나로 통합(필터 + 국가 플래그 + 시술명 태그). 블로그를 "시술 가이드"로 리프레이밍.

**Phase 4 — 안내**: 지도, Inquiry Form, 챗봇은 기존 유지. 이 영역은 유저가 오프라인으로 넘어가기 직전의 보조 안내 역할.

**실행 로드맵 (진행 시)**

| 단계 | 내용 | 비고 |
|------|------|------|
| Phase 0: 기반 | 하드코딩 콘텐츠 어드민화 + 측정 설계 (UTM 어트리뷰션, 크로스 세션 전환 이벤트) | 어떤 개선이든 측정 없이는 효과 검증 불가 |
| Phase 1: 핵심 개편 | 4-Phase 구조 적용, Hero 리디자인, 캐시백 독립, 인기 시술 섹션, Recommendation 간소화, 리뷰/블로그 통합 | 공수 큼 — 진입 트래픽 확대와 병행 시 착수 권장 |
| Phase 2: 콘텐츠 강화 | 가격 비교 가이드, Hero 언어별 분기, 카테고리 탭 언어별 순서 | Phase 1 효과 검증 후 |
| Phase 3: 고도화 | 클리닉 특성 태그, 리뷰 시술명 태그, Before/After 갤러리 | 콘텐츠 수급 후 |

**권고**: 현시점에서는 **진입 트래픽 확대(GNB 승격 A/B, 메인 카드 리디자인)를 우선 실행**하고, 트래픽이 일정 수준(예: 월 2,000세션 이상) 이상으로 올라온 시점에 페이지 내부 구조 개편을 본격 착수하는 것이 리소스 대비 효율적입니다. 당장은 Phase 0(어드민화 + 측정 설계)만 선행해 두면, 이후 트래픽이 확보되었을 때 빠르게 개편에 착수할 수 있습니다.
