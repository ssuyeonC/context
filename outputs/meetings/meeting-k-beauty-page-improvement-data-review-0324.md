## 1. 데이터 리뷰 — 전환 퍼널

### 1-1. 전체 퍼널 숫자

**Growth Manager**: 1개월간의 전환 퍼널을 정리하면 다음과 같습니다.

```
K뷰티 페이지 조회        873 세션 (100%)     ← GA page_view 기준
        │
        │  31.8% 인터랙션
        ▼
페이지 내 클릭 발생       278 세션 (31.8%)    ← DB general_click_event 기준
        │
        │  59.4% 스팟 카드 클릭
        ▼
스팟 카드 클릭            165 세션 (18.9%)
        │
        │  7.9% 구매
        ▼
스팟 구매 완료             13 세션 (1.49%)    ← GA 같은 세션 내 purchase
```

총 GMV: 1,453,207원 (건당 평균 약 11.2만원)

### 1-2. 데이터 신뢰성에 대한 비판적 검토

**PM**: 잠깐, 이 퍼널을 그대로 읽기 전에 **데이터 소스가 섞여 있다는 점**부터 짚어야 합니다.

| 구간 | 데이터 소스 | 한계 |
|------|-----------|------|
| 조회 (873) | GA `UserEventsAcquiredFromAll` | page_view 이벤트 기준, 봇 트래픽 필터 여부 불확실 |
| 클릭 (278) | DB `general_click_event` | **클릭 이벤트를 기록하는 코드가 삽입된 요소에서만 캡처** — 페이지 내 모든 인터랙션을 의미하지 않음 |
| 구매 (13) | GA `domain_transaction_id` 기준 | **같은 세션** 내 구매만 잡힘 — 재방문 구매 미포함 |

**Growth Manager**: PM 말이 맞습니다. 특히 두 가지 문제가 큽니다.

**첫째, "68.2%가 아무것도 안 하고 이탈"이라는 해석은 과장일 수 있습니다.** `general_click_event`는 모든 클릭을 기록하는 게 아니라, 이벤트 트래킹이 심어진 요소만 캡처합니다. 스크롤, 탭 전환, 카테고리 클릭 같은 인터랙션이 기록되지 않을 수 있어요. 실제 이탈률은 GA의 bounce rate로 별도 확인이 필요합니다.

**둘째, 구매 전환 13건(1.49%)은 "같은 세션 내" 조건이라 과소 측정입니다.** 크로스 세션 데이터를 보면, K뷰티 페이지를 본 666명 중 219명(32.9%)이 같은 기간 내 스팟 상세를 방문했어요. 뷰티 시술은 고관여 의사결정이라 "오늘 보고 내일 예약" 패턴이 일반적입니다. 같은 세션 CVR 1.49%만 보면 이 페이지의 기여를 과소평가하게 됩니다.

**PO**: 좋은 지적입니다. 그러면 **이 데이터로 확실하게 말할 수 있는 것과 없는 것**을 구분합시다.

| 확실하게 말할 수 있는 것 | 말할 수 없는 것 |
|------------------------|---------------|
| 월 873 세션은 허브 페이지로서 트래픽이 적다 | 실제 이탈률이 68%인지 (다른 인터랙션 미캡처) |
| 스팟 카드를 클릭한 유저 중 구매 전환은 낮다 | 재방문 구매를 포함한 실제 기여 전환율 |
| 구매 스팟은 뷰티 카테고리에 정확히 일치 | 이 구매가 K뷰티 페이지 "덕분에" 일어났는지 (어트리뷰션) |
| 크로스 세션으로 32.9%가 스팟 상세를 방문 | 그 스팟 상세 방문이 K뷰티 페이지 경유인지 직접 유입인지 |

## 2. 데이터가 기존 합의에 미치는 영향

### 2-1. Q1 재검증 — "개선이 맞는가?"

**Growth Manager**: 3/22 회의에서 "메인 → K-beauty CTR 10%, 스팟 클릭율 30~33%"를 근거로 "개선은 맞되 진입이 문제"라고 결론냈었습니다. 새 데이터를 대입하면:

| 3/22 기준 | 새 데이터 | 차이 원인 |
|----------|----------|----------|
| 스팟 클릭율 30~33% | 18.9% (GA 모수 기준) | 3/22는 클릭 세션(278) 대비 → 실제 PV 세션(873) 대비로 바뀜 |
| — | 59.4% (클릭한 세션 중) | 클릭 행동을 한 유저 한정이면 여전히 높음 |

**PM**: 이건 중요한 차이입니다. **"들어온 유저의 30%가 스팟을 클릭한다"는 3/22 논의의 전제가 흔들립니다.** 실제로는 873명 중 165명, 즉 **18.9%**입니다. "페이지 내 전환력이 기반이 있다"는 판단을 유지할 수 있는지 재검토가 필요해요.

**Marketer**: 하지만 **18.9%도 나쁜 숫자는 아닙니다.** 5명 중 1명이 스팟을 클릭한다는 건, 페이지 방문자의 유입 인텐트가 뷰티 시술 탐색과 어느 정도 일치한다는 뜻이에요. 문제는 "나머지 81%가 왜 클릭하지 않았는가"인데, 이건 3/22에서 논의한 **정보 과부하(18개 섹션), 스토리라인 부재** 문제와 직접 연결됩니다.

**PO**: Q1 결론은 변경 없이 유지합니다. 다만 **"페이지 내 전환력이 기반이 있다"는 근거를 30~33%에서 18.9%로 하향 조정**합니다. 개선의 필요성은 오히려 더 명확해졌어요.

### 2-2. Q2 재검증 — "진입 확대의 우선순위"

**Growth Manager**: 873 세션이라는 절대 숫자를 봅시다. **한 달에 873 세션**이면 하루 약 30 세션입니다. 뷰티를 전략 카테고리로 밀고 있는 상황에서 이건 심각하게 적어요.

**PM**: 3/22에서 진입 확대를 3트랙(A: GNB 승격, B: 메인 카드 리디자인, C: 맥락적 진입점)으로 잡았는데, 이 숫자를 보면 **A(GNB 승격)의 우선순위를 더 올려야 할 수 있습니다.** 메인 카드 리디자인(B)은 메인 페이지에 이미 있는 진입점을 개선하는 거라 점진적 효과인 반면, GNB 승격은 **모든 페이지에서 진입 가능**하게 만드는 구조적 변화이니까요.

**Marketer**: 동의합니다. 그런데 **방안 C(맥락적 진입점)도 데이터로 보면 재평가가 필요합니다.** 크로스 세션 데이터에서 K뷰티 유저 666명 중 219명(32.9%)이 같은 기간 내 스팟 상세를 방문했어요. 이건 **"스팟 상세 → K뷰티 역유입" 경로가 유효하다**는 간접 증거입니다. 지금 개발 중인 **랜딩페이지 진입점 노출 기능(COM-2368)**도 이 맥락적 진입에 해당하고요.

**PO**: 흥미로운 포인트입니다. COM-2368은 "스팟 상세에서 랜딩페이지로"의 경로인데, 이것과 유사하게 "스팟 상세에서 K뷰티 페이지로"의 경로도 방안 C(P1-3)에 포함되어 있습니다. **두 기능이 스팟 상세 하단이라는 동일한 공간을 놓고 경쟁할 수 있어요.** 이건 설계 시 같이 고려해야 합니다.

**PM**: 맞습니다. 스팟 상세 하단에 "이 스팟의 이벤트 보기"(랜딩페이지 진입점)와 "더 많은 뷰티 시술 보기"(K뷰티 CTA)가 함께 노출되면, 유저 인지 부하가 올라갈 수 있어요. **COM-2368이 먼저 출시되니, P1-3의 CTA 위치와 형태를 COM-2368과 충돌하지 않게 설계해야 합니다.**

### 2-3. Q3 재검증 — "내부 개선의 초점"

**Growth Manager**: 퍼널에서 **가장 큰 drop-off가 어디인지**를 데이터로 봅시다.

```
873 → 165 (스팟 클릭): 708명 이탈, 전환율 18.9%  ← 가장 큰 손실
165 → 13  (구매):       152명 이탈, 전환율 7.9%   ← 두 번째 손실
```

**첫 번째 구간(조회 → 스팟 클릭)에서 708명이 사라집니다.** 이건 3/22에서 진단한 "스토리라인 부재, 정보 과부하"와 직결됩니다. Phase 2의 구조 재편(P2-1)과 Trust 통합(P2-2)이 이 구간을 개선하는 태스크예요.

**Marketer**: 근데 여기서 비판적으로 봐야 할 게 있습니다. **708명이 "이탈"한 건지, "스팟 외 콘텐츠(블로그, 리뷰, 지도)를 소비하고 나간 건지"를 구분해야 해요.** K뷰티 페이지의 목적이 "스팟 구매"만은 아니잖아요. 뷰티 정보 탐색, 블로그 소비, 캐시백 인지 — 이것들도 페이지의 역할입니다. 스팟 클릭률만으로 페이지 실패를 판단하면 안 됩니다.

**PM**: 좋은 포인트입니다. 다만 **회사가 뷰티를 밀고 있는 이유는 거래액(GMV) 때문이지 콘텐츠 소비 때문이 아닙니다.** 블로그 소비도 결국 스팟 구매로 이어져야 의미가 있어요. 물론 즉시 전환이 아니라 인지 → 재방문 → 구매의 간접 경로도 있지만, **측정 가능한 직접 전환이 1.49%라는 건 사실입니다.**

**Growth Manager**: 두 번째 구간(스팟 클릭 → 구매, 7.9%)도 따져봅시다. 이건 사실 **K뷰티 페이지의 문제가 아니라 스팟 상세 페이지의 문제**일 수 있어요. 스팟 상세에서 구매를 안 하는 건 가격, 옵션, 리뷰, 예약 UX 등 스팟 상세 자체의 이슈이지, K뷰티 페이지가 개선한다고 좋아지는 구간이 아닙니다.

**PM**: 맞습니다. **단, 하나의 예외가 있어요.** COM-2368(랜딩페이지 진입점)이 나오면, 스팟 상세에서 이벤트(캐시백, 할인)를 인지시킬 수 있고, 이건 두 번째 구간의 전환율을 올릴 수 있는 수단이에요. K뷰티 페이지 자체가 아니라 스팟 상세에서의 개선이지만, 뷰티 전환 퍼널 전체로 보면 연결됩니다.

**PO**: 정리합니다. Q3 내부 개선은 **첫 번째 구간(조회 → 스팟 클릭, 18.9%)을 올리는 것이 주 목표**입니다. 두 번째 구간은 K뷰티 페이지의 scope가 아니라 스팟 상세/랜딩페이지 진입점(COM-2368)의 scope로 봅니다.

## 3. 구매 스팟 분석과 콘텐츠 시사점

### 3-1. 어떤 스팟이 전환되고 있는가

**Growth Manager**: 구매된 스팟을 보면:

| 스팟 | 세션 | GMV | 카테고리 |
|------|------|-----|---------|
| 아비쥬의원 명동 글로벌점 | 2 | 30,000원 | 피부과 |
| THE SEOL:HA더설하 한옥 | 1 | 737,000원 | 헤드스파/바디 |
| 브로우돌 강남 반영구 | 1 | 225,000원 | 반영구 |
| 왁싱앤라인 홍대점 | 1 | 110,000원 | 왁싱 |
| 박준뷰티랩 강남점 | 1 | 60,000원 | 헤어 |
| 휩드 | 1 | 60,000원 | DIY 코스메틱 |
| 기타 7건 | 각 1 | 231,207원 | 혼합 |

**Marketer**: 두 가지가 눈에 띕니다.

첫째, **카테고리가 넓게 분포합니다.** 피부과, 헤드스파, 반영구, 왁싱, 헤어, DIY — 특정 카테고리에 집중되지 않아요. 이건 K뷰티 페이지가 **다양한 뷰티 니즈의 유저를 끌어들이고 있다**는 뜻이기도 하지만, 반대로 **"킬러 카테고리"가 없다**는 뜻이기도 합니다.

둘째, **THE SEOL:HA 73.7만원이 전체 GMV의 절반**입니다. 13건 중 1건이 전체 GMV의 50.7%를 차지해요. **이 1건을 빼면 평균 건당 GMV가 5.9만원으로 떨어집니다.** 데이터가 너무 적어서 평균이 의미가 없어요.

**PM**: Marketer 말대로 **샘플 사이즈(13건)가 너무 작아서 이 데이터로 콘텐츠 전략을 세우기는 어렵습니다.** "피부과를 밀어야 한다"든 "헤어를 밀어야 한다"든, 13건으로는 아무것도 결론 내릴 수 없어요.

**Growth Manager**: 동의합니다. 다만 **DB 클릭 이벤트 기준(24시간 내 구매, 39건)**을 교차 확인하면 추가적인 시사점은 있습니다.

| 스팟 | DB 기준 세션 | DB 기준 주문 |
|------|------------|------------|
| 모덴리 홍대 반영구 | 3 | 9 |
| 순시키헤어 홍대 본점 | 2 | 8 |
| 아비쥬의원 강남 글로벌점 | 2 | 13 |
| 명동 네일아트 프로뷰티 | 2 | 15 |
| 홍대 초정밀 피부명가 | 2 | 14 |
| 앤봄클리닉 홍대 피부미용 | 2 | 11 |

**PM**: 잠깐, 이 숫자는 **"K뷰티 페이지에서 클릭한 세션의 member_code가 24시간 내 구매한 것"**이지, "K뷰티 페이지 때문에 구매한 것"이 아닙니다. 이 유저들이 K뷰티 페이지를 경유해서 구매한 건지, 아니면 원래 그 스팟을 알고 있었고 K뷰티 페이지는 그냥 스쳐간 건지 **어트리뷰션을 알 수 없어요.**

**PO**: 정확합니다. 이 데이터로 할 수 있는 건 **"K뷰티 유저가 어떤 카테고리에 관심이 있는가"의 경향성 파악 정도**이고, 이를 근거로 K뷰티 페이지의 카테고리 노출 순서나 Quick Match(Embedded Step Wizard)의 기본 옵션 배치에 반영할 수 있습니다.

## 5. COM-2368(랜딩페이지 진입점)과의 시너지

**PO**: 현재 진행 중인 COM-2368과의 관계를 정리합시다. K뷰티 페이지 개선과 COM-2368은 **"뷰티 유저의 전환 퍼널을 개선한다"는 동일한 목표**를 다른 접점에서 달성하려는 시도입니다.

```
[K뷰티 페이지 개선]          [COM-2368 랜딩페이지 진입점]
  K뷰티 페이지 → 스팟 클릭        스팟 상세 → 이벤트 인지
  (첫 번째 구간 개선)              (두 번째 구간 개선)
```

**Growth Manager**: 이 두 기능이 **직접적으로 시너지를 낼 수 있는 시나리오**가 있습니다.

1. K뷰티 페이지에서 스팟 A를 클릭 → 스팟 A 상세에서 이벤트(랜딩페이지) 진입점 노출 → 이벤트 인지 → 구매 동기 강화
2. 반대로, 랜딩페이지에서 스팟 B를 발견 → 스팟 B 상세에서 "더 많은 뷰티 시술"(K뷰티 CTA) → K뷰티 페이지 → 교차 탐색

**PM**: 다만 2번은 P1-3(맥락적 진입점, P2 우선순위)이 구현되어야 가능한 시나리오입니다. COM-2368이 먼저 나가면 1번 시나리오만 가능하고, 그것만으로도 **K뷰티 → 스팟 클릭 → 이벤트 인지 → 구매**의 풀 퍼널이 만들어집니다.

**PO**: COM-2368 출시 후 이 풀 퍼널의 전환율을 측정하면, K뷰티 페이지 개선의 **두 번째 구간(스팟 클릭 → 구매) 기여도**를 간접적으로 파악할 수 있겠네요.
