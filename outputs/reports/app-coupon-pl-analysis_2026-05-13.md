## 3. 유저 그룹 분류

### 모그룹(U)
발급된 앱 전용 쿠폰을 사용하여 앱 결제를 진행한 유저 전체 (5/13 기준 **923명**).

### 분류 정의

| 그룹 | 분류 | 정의 |
|---|:---:|---|
| **A** | 손실군 | 발급 이전 가입 + 발급 이전 앱 결제 이력 **있음** |
| **B-1** | 손실군 | 발급 이전 가입 + 앱 결제 이력 없음 + 베이스라인 **내부** |
| **B-2** | **이득군** | 발급 이전 가입 + 앱 결제 이력 없음 + 베이스라인 **외부** |
| **C-1** | 손실군 | 발급 이후 가입 + 베이스라인 **내부** |
| **C-2** | **이득군** | 발급 이후 가입 + 베이스라인 **외부** |

**베이스라인**: 일 71명 (4/1~4/20 평균 첫 앱 결제 수, 쿠폰 정책 도입 이전)
**증분(이득군)** = 기간 동안 실제 첫 앱 결제 수 - 베이스라인 × 기간일수

### 그룹별 인원 (5/13 기준)

| 그룹 | 분류 | 인원 | 비중 | 산출 방식 |
|---|:---:|---:|---:|---|
| **A** | 손실군 | 169 | 18.3% | DB 직접 측정 (EXISTS 쿼리) |
| **B-1 + C-1** | 손실군 (베이스라인 내부) | ~469 | 50.8% | 차감 (U - A - 이득군) |
| **B-2 + C-2** | **이득군** (마케팅 타겟) | **285** | **30.9%** | 베이스라인 대조 추정 |
| **모그룹 U 합계** | | 923 | 100% | DB 직접 측정 |

**손실군 합계**: A + (B-1 + C-1) = 638명 (69.1%)
**이득군 합계**: B-2 + C-2 = 285명 (30.9%)

### 손익 공식

```
① 손실        = (A + B-1 + C-1)의 쿠폰 할인액         → 회수 불가
② 마케팅 투자 = (B-2 + C-2)의 쿠폰 할인액              → 의도된 비용
③ 즉시 이득   = (B-2 + C-2) 결제건 GMV × 이익률       → 단기 회수
④ LTV 이득    = (B-2 + C-2) × 인당 incremental GMV × 이익률   → 장기 회수

순효과 = ③ + ④ - ② - ①
```

> 손실군의 결제 이익은 정책 유무와 무관하게 발생했을 자연 발생분이므로 counterfactual 기준 카운트 안 함.

### Break-even 계산

```
순효과 = 0인 이익률 M에서:
```

→ **이익률 break-even ≈ 9.73%**. 실측 10.35%에서는 **~0.62%p의 안전 마진**.
