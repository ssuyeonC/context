### **영역 3. 파트너와의 관계 — Type D 가설 검증 (10~15분)**

| **#** | **질문** |
| --- | --- |
| 3-1 | 파트너가 직접 예약 확정을 하는 경우가 있나요? 어떤 파트너가 잘 하고, 어떤 파트너가 안 하나요? |
|  |   • 메디큐브는 파트너 어드민을 많이 쓰는데?
      ◦ 승인이 너무 느려서 컴플레인 → 어드민 제안 → 전환됨
      ◦ 메디큐브는 파트너 어드민을 사용하는 분이 한 분이 있어서 더 편하게 전환된듯
      ◦ 파트너 어드민은 한 분이 관리하시는 게 좋음 |
| 3-2 | 확정을 안 하는 파트너에 대해서, 직접 처리하도록 안내해본 적 있으세요? |
|  |   • 뷰티블라썸 - 파트너 어드민을 선제안, 단톡방 없음
  • 단톡방 - cx팀 + 파트너 / 제휴팀 관리자 + 파트너 |
| 3-3 | *(안내한 적 있다면)* 어떤 방식으로 안내하셨어요? 반응은 어땠나요? |
|  |   • 뷰티블라썸 - 제안에 대해 협조적이었음
      ◦ 파트너 담당자 연령대도 중요함 |
| 3-4 | *(안내한 적 없다면)* 안내하지 않은 특별한 이유가 있나요? |
|  |  |
| 3-5 | 파트너가 어드민 사용법을 물어본 적이 있나요? 있다면 어떤 걸 물어봤나요? |
|  |   • 특별히 없었는데 버그 리포트 해주시는 케이스는 있었음 |
| 3-6 | 파트너가 알림톡을 받고 있는지 알고 계세요? 알림톡을 안 받는 파트너가 있다면 왜 그런지 아시나요?
알림톡을 받도록 설정하는 어떤 기준이 있는 것 같은데, 그 기준이 무엇인가요? |
|  |  |
| 3-7 | 만약 파트너가 직접 확정을 많이 하게 되면, 본인 업무가 어떻게 달라질 것 같으세요? |
