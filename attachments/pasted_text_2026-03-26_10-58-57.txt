## SUMMARY

- K-Beauty 페이지는 월 873세션, 스팟 클릭율 18.9%로 뷰티 전략 카테고리의 허브 역할을 하기엔 부족함
- 리서치 결과, 뷰티 페이지의 타겟은 ****"시술을 받기 위해 한국에 오는" **고관여 목적형 방문자**
- 현재 페이지는 **K뷰티를 소개**하는 영역과 **특정 기준으로 스팟을 병렬적으로 나열**하는 구조로 설계되어 있음
    - 이들의 의사결정 흐름과 불일치

## 1. 배경

### 전략적 맥락

- 회사가 뷰티를 전략 카테고리로 밀고 있는 상황에서, K-Beauty 페이지는 뷰티의 **유일한 허브 페이지**
- 개별 스팟 상세로의 진입점 중 하나이자, "한국에서 뷰티 시술을 받고 싶다"는 유저의 탐색 시작점
    - 스팟 리스트 페이지 대비 K뷰티 시술에 대한 크리에이트립만의 강점 (크리에이트립을 이용하는 것이 이득) 어필 가능

### 현재 상태 (3/24 데이터 기준, dba-mcp 활용)

```
K뷰티 페이지 조회      873 세션/월 (일 약 30)
       │
       │  18.9% 스팟 클릭
       ▼
스팟 카드 클릭          165 세션
       │
       │  7.9% 구매
       ▼
스팟 구매 완료           13 세션 (CVR 1.49%)
```

- **진단**
    - **월 873세션**은 허브 페이지로서 절대적으로 부족
    - **81%의 유저(708명)가 스팟을 클릭하지 않고 이탈** — 가장 큰 손실 구간
    - 같은 세션 CVR 1.49%는 과소 측정 가능성
        - 상품 탐색 기간이 긴 고관여 상품 특성 상 조회와 결제가 동일 세션에 결제가 일어나기 어려움
    - 시술 추천(8단계 모달) CTR은 **1.4%** — 사실상 사용되지 않는 기능

### 구조적 문제

| 문제 | 현재 상태 | 영향 |
| --- | --- | --- |
| 정보 과부하 | **18개 섹션**이 타입별로 산발 배치 | 유저가 "다음에 뭐가 나올지" 예측 불가, 이탈 유발, 정보 과부하 |
| 스토리라인 부재 | 블로그 3곳, 리뷰 3곳 분산 | 같은 유형 콘텐츠가 반복 — 유저가 차이를 인지 못함 |
| 추천 진입 장벽 | 8단계 모달, All 탭에서만 노출,  | CTR 1.4%, 개인화 경험 사실상 미제공 |
| 기술 부채 | 블로그 배너 14개 언어 하드코딩, Trust 숫자 하드코딩 | 콘텐츠 교체가 개발 배포에 의존 — 마케팅팀 자체 운영 불가 |

## 2. 고객 인사이트

> Gemini 딥 리서치 활용 (출처: 주요 레딧 포스트, 리서치 자료([야놀자 - 한국 메디컬 투어리즘](https://www.yanolja-research.com/insight/view/596?lang=kr)) 등 50+)
> 

### 외국의 한국 피부미용 시술 수요자는 초보자가 아니라 고관여 소비자다

- 현재 페이지는 "한국에서 뷰티 시술을 받을 수 있어요"와 시술 자체를 설명하는 데 지면의 상당 부분 활용
- 수요자는 피부 시술과 관련된 정보의 재생산이 빠른 커뮤니티와 SNS 등을 통해 시술에 대한 기본 정보 인지

**전제 1 — 피부 미용이 여행의 '목적'이다**

- 피부미용을 위해 한국을 찾는 외국인 관광객은 "한국 여행 온 김에 피부 미용을 받는다"가 아니라 **"피부 미용을 받기 위해 한국에 방문한다"**에 가까움
    - 피부과 진료가 외국인 환자 방문의 **56.6%**로 압도적 1위 (2024년 117만 명)
    - 리쥬란 엠보싱 회복 기간을 고려해 여행 일정 자체를 조정하는 수준
    - "한국에서의 울쎄라는 비행기 값을 뽑는 시술"이라는 인식

→ **"여기서 뷰티 시술이 가능해요"는 이미 알고 있는 정보. 인지 단계를 길게 가져갈 필요 없음.**

**전제 2 — 고객은 이미 '무엇을 받아야 하는지' 안다**

- 상품 구매를 결정하기 전에 커뮤니티와 SNS 등을 통해 시술에 대한 충분한 정보를 학습하는 경향
- 리쥬란 종류별 차이(힐러/HB Plus/S/I)를 구분하고 목적에 맞게 선택
- 콤비네이션 테라피(울쎄라+리쥬란, 포텐자+엑소좀)를 **고객이 먼저 제안**

→ **"피부과란 무엇인가"를 설명하지 말고, "이 시술을 어디서, 얼마에, 어떤 조합으로 받을 수 있는가"에 빠르게 도달하는 구조가 필요**

**전제 3 — 국가별로 고객 여정이 완전히 다르다**

| 국적 | 핵심 동기 | 의사결정 기준 | 선호 시술 |
| --- | --- | --- | --- |
| 일본 | 가성비 + 빠른 시술 | 가격, 소요 시간 | 보톡스, 레이저, 점 빼기 |
| 중국 | 정품 신뢰 + 프리미엄 | 정품 인증, 의료진 경력 | 울쎄라, 써마지 |
| 미국/유럽 | 과학적 근거 | 성분 원리, 임상 결과 | 리쥬란, 쥬베룩 |
| 동남아 | K-뷰티 동경 | 비주얼 결과, 트렌드 | 리쥬란, 엑소좀 |

→ **단일 구조로 모든 국가를 커버하되, 콘텐츠 레이어에서 국가별 차별화가 가능한 구조까지 설계되면 바람직함**

### 추가 인사이트

| 인사이트 | 제품 시사점 |
| --- | --- |
| 가격 투명성이 신뢰의 전제 조건 (레딧으로 시세 파악, 서면 견적서 요구) | 가격 정보를 숨기면 역효과.
시술별 시세 범위, 정품 인증 정보가 핵심 |
| 공장형 vs 부티크 병원 구분이 이미 상식 | 병원 유형/특화 분야 태깅이 차별화 포인트 |
| 콤비네이션 시술이 객단가 상승의 열쇠 | 단일 시술 나열 
**→ 목적별 조합 패키지 추천** |
| 차세대 시술(쥬베룩, 엑소좀) 키워드 급부상 | 선제적 콘텐츠 확보로 얼리어답터 선점 기회 |

## 3. 해결 방향: 4-Phase 스토리라인 기반 재설계

### 설계 원칙

1. **인지는 짧고 강하게, 탐색으로 빠르게 전환** — 목적형 방문자에게 "왜 크리에이트립인가"만 전달
2. **시술명 → 스팟 매칭을 최단 경로로** — 유저가 "리쥬란"을 알고 왔으면 2클릭 이내로 해당 스팟에 도달
3. **탐색 진입점을 복수로 제공** — 인기 시술로, 간소화된 개인화 추천 기능으로, 카테고리로 — 여러 경로
4. **18개 → 10개 섹션으로 축소** — 정보 과부하 해소, 스토리라인 명확화

### 전체 페이지 구조

- 와이어프레임: https://wireframes-lkb.pages.dev/#k-beauty-redesign

```

Phase 1: 인지 — "여기서 예약하면 이득이다"
├─ [1-A] Hero (비주얼 + 카피 + Social Proof 인라인)
├─ [1-B] Beauty Cashback (독립 섹션)
└─ [1-C] Trust — Why Creatrip (3카드)

Phase 2: 탐색 — "나에게 맞는 시술과 클리닉 찾기"
├─ [2-A] 실시간 인기 시술
├─ [2-B] Embedded Step Wizard (시술→예산→지역)
├─ [2-C] Category Tab + Spots
└─ [2-D] 가격 비교 가이드

Phase 3: 검증 — "실제 결과와 전문성으로 신뢰 확인"
├─ [3-A] 통합 리뷰 (필터 + 국가 플래그 + 시술명 태그)
└─ [3-B] 시술 가이드 (블로그 리프레이밍)

Phase 4: 행동 — "예약하기"
├─ [4-A] 지도
└─ [4-B] Inquiry Form

[Floating] 챗봇
```

### Phase 3: 검증 — "실제 결과와 전문성으로 신뢰 확인"

![image.png](attachment:5fc6eeb7-bb68-43e6-a550-3da300bd83c4:image.png)

> 실제 결과와 전문성으로 신뢰 확인
근거: 리서치에서 검증 행동이 후기 읽기를 넘어 정품 인증, 장비 확인, 생물학적 원리 질문까지 포함
> 

**[3-A] 통합 리뷰 섹션**

- 기존 3개(Review Choice Groups, Recent Reviews, Photo Reviews) → **하나로 통합**
- 필터(전체/포토/영상) + **국가 플래그 아이콘** + **시술명 태그**
- 국가 플래그: 리서치에서 고객이 "같은 국가 사람의 후기"를 특히 신뢰
- 시술명 태그: "리쥬란 후기"를 찾는 유저가 스팟 이름만으로는 시술 내용을 알 수 없음

**[3-B] 시술 가이드 (블로그 리프레이밍)**

- "블로그" → "시술 가이드"로 명칭 변경 — 검증 단계 성격에 부합

## 5. 기대 효과

### 정량적 목표

| 지표 | 현재값 | 목표값 |
| --- | --- | --- |
| K-beauty 내 스팟 클릭율 | 18.9% | **25~30%** |
| 시술 추천 참여율 | 1.4% | **5~10%** |
| 메인 → K-beauty 진입율 | ~10% | **15~20%** |
| 페이지 이탈율 | 미측정 | 베이스라인 대비 **-15%** |
| 스크롤 깊이 | 미측정 | **50% 이상 유저가 Phase 3 도달** |
| 크로스 세션 전환율 (7일) | 미측정 | **측정 시작** |

### 정성적 기대

- **객단가 상승**: 콤비네이션 패키지 발견 경로 확보 → 조합 시술 전환
- **외부 유입**: 시술 비교 가이드가 레딧/SNS 공유 포맷과 일치 → 오가닉 유입 경로 확보
- **운영 자율성**: 어드민화로 마케팅팀이 콘텐츠를 직접 관리 가능 → 개발 의존 탈피
